The big idea

Local fixes, repeated

Bubble Sort is slow for large data, but it is one of the clearest ways to see how adjacent comparisons can gradually create order.

i.

Compare neighbours

Only adjacent values are inspected at each step.

ii.

Swap if reversed

If the left value is larger, the pair changes places.

iii.

Stop early

If a full pass has no swaps, the array is already sorted.

When to use Bubble Sort

Bubble Sort is mainly useful for teaching adjacent swaps and early-exit detection, not for production sorting.

  • Classroom demonstrations of pairwise comparison.
  • Checking whether a list is already sorted with an early-exit pass.
  • Very small toy inputs where clarity is more important than speed.
