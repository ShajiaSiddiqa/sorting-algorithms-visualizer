When to use Radix Sort

Use Radix Sort for fixed-width integer-like keys when stable digit passes are cheap.

  • IDs, phone-number-like numeric keys, timestamps, and byte sequences.
  • Large integer arrays where comparison sorting overhead is high.
  • Systems that can trade extra memory for near-linear sorting.
