When to use Counting Sort

Use Counting Sort when keys are integers in a small known range. It is not comparison-based, so it can beat O(n log n).

  • Exam scores, ages, ratings, categories, or byte values.
  • As a stable subroutine inside Radix Sort.
  • Histograms and frequency tables where counts are already useful.
